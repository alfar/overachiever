Overachiever

Gamification for websites and web applications made easy. Create, track, and award achievements, badges, or trophies.

Create

Create achievements for your website with Overachiever. Our easy-to-use interface makes it easy to design and create achievements, without needing to know about web programming or scripting.

Track

It just takes a simple JSON request to track activity on an achievement. Let Overachiever figure out when your users have earned achievements. You can focus simply on what activities earn an achievement, and what to do when someone earns it.

Award

Overachiever awards achievements when users have the activity to earn them. With a simple JSON request, you can get a single user's list of achievements, or check to see if a user has a particular achievement. Making this easier lets you do more with that functionality — whether you just want to display a badge, or open up new functionality.
